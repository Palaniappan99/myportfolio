PROJECTS

Integration of Wireless sensors for Leak Detection

  • This prototype is modelled to identify leaks in the industry pipeline setup using wireless sensors like Pressure and Temperature sensors
  • Before building the prototype, the model was designed using Fusion 360 and Simulated and Calibrated using COMSOL software.
  • Prototype is designed with small wireless nodes distributed along the pipelines, pressure sensors are installed next to the manually created leak nodes. The leak is detected using the pressure point analysis method.
  • Various pressure datas are obtained from the fluid flow networks in the pipes and its variations are analysed using Bernoulliâ€™s equation, to monitor leaks in pipes.
  • A cloud storage is used to store the historical data and the data is transmitted using ESP8266 Wi-Fi module.

On-Board Weighing System for Heavy Vehicles

  • This prototype is modelled to weigh the load on back of the truck without unloading it.
  • A series of Load cells (Strain Gauges), conncetd using a HX711 module, are placed in the axles of the truck. Based on the bending of the axle, the load cell reads the weight from different axle.
  • These weight data from different axles are then processed to give the accurate weight using Arduino. This weight is then transmitted to the LCD display on the truck cabin.
  • The weight reading is also transmittedto the Blynk mobile app using ESP8266 Wi-Fi module.

Time Study for Ergonomic Environment

NVA is a type of work where the client does not pay money but is required to perform NVA and Waste work is a type of work where it is not neccessary and also client doesn't provide money, like transporting of goods from one work station to other.
